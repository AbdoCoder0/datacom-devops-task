# Datacom DevOps Simulation - The Big Red Button 🔴

## Project Overview
This repository contains the infrastructure and automation workflow for the "Big Red Button" project. The primary goal is to enable repeatable and reproducible infrastructure deployments to ensure a smooth end-user experience.

## Objective
To bridge the gap between software development and hardware configuration by implementing Infrastructure as Code (IaC) principles. This repository serves as the central hub for the CI/CD pipeline.

## Technologies Used
* **Version Control:** Git & GitHub
* **CI/CD:** GitHub Actions
* **Cloud Provider:** Microsoft Azure
* **Automation:** YAML workflows
